# نشر لوحة متابعة البرنامج الحكومي لحماية الأسرة على GitHub Pages

هذا المجلد مخصص للنشر للفريق كنسخة **قراءة فقط** (تصميم «مركز القيادة» الداكن).

## الملفات

- `index.html`: نسخة العرض للفريق (قراءة فقط، بلا أزرار تعديل، تُحمّل الحالة من `data.json`).
- `data.json`: بيانات الحالة التي تصدّرها من نسخة الإدارة.
- `assets/logo.png`: الشعار الرسمي المستخدم في الترويسة.

## طريقة التحديث السريعة

1. افتح **نسخة الإدارة** المحلية (القابلة للتعديل):
   `dashboard.command-center.html`
2. عدّل الحالات والنسب والمؤشرات والتقارير.
3. اضغط زر الشريط الجانبي: **«⬇ تصدير بيانات النشر»** لتنزيل `data.json`.
4. استبدل ملف `github-pages/data.json` بالملف الجديد، ثم:
   ```bash
   cd github-pages
   git add data.json
   git commit -m "Update data.json"
   git push origin main
   ```
5. ينتقل التحديث تلقائيًا إلى رابط الفريق خلال ثوانٍ أو دقائق.

> ملاحظة: عند تغيّر التصميم نفسه (لا البيانات)، انسخ `dashboard.command-center.html` إلى `index.html` ثم أعد تطبيق وضع القراءة فقط (تحميل `data.json` وإخفاء أدوات التحرير).
