# نشر لوحة متابعة البرنامج الحكومي لحماية الأسرة على GitHub Pages

هذا المجلد مخصص للنشر للفريق كنسخة قراءة فقط.

## الملفات

- `index.html`: نسخة العرض للفريق، بدون أزرار تعديل.
- `data.json`: بيانات الحالة التي تصدرها من نسخة الإدارة.

## طريقة التحديث السريعة

1. افتح نسخة الإدارة المحلية:
   `dashboard.html`
2. عدّل الحالات والنسب والتقارير.
3. اضغط زر: `تنزيل بيانات النشر data.json`.
4. ارفع ملف `data.json` الجديد في مستودع GitHub بدل الملف القديم.
5. ينتقل التحديث تلقائيًا إلى رابط الفريق بعد ثوانٍ أو دقائق.

## رابط الفريق

بعد تفعيل GitHub Pages سيكون الرابط عادة بهذا الشكل:

`https://USERNAME.github.io/REPOSITORY/`

استبدل USERNAME و REPOSITORY باسم حسابك واسم المستودع.
